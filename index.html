Traffic Counter App

Browser-based pedestrian and vehicle counting dashboard

Project Overview

This web app is being developed to detect and count pedestrians and vehicles from video footage. The goal is to create a simple real-time dashboard that can support traffic analysis, facility planning, and space usage decisions.
